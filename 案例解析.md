## з¬¬дёҖйҳ¶ж®өпјҡScanner иҜҚжі•жү«жҸҸ

**иҫ“е…Ҙпјҡ** жәҗз Ғеӯ—з¬ҰдёІ
**иҫ“еҮәпјҡ** Token еҲ—иЎЁ

Scanner йҖҗеӯ—з¬ҰиҜ»еҸ–пјҢиҜҶеҲ«иҜҚжі•еҚ•е…ғпјҡ

```
жәҗз Ғеӯ—з¬Ұ           иҜҶеҲ«з»“жһң
в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
v a r              вҶ’ Token(VAR,      "var",   null, line=1)
  a                вҶ’ Token(IDENTIFIER,"a",    null, line=1)
    =              вҶ’ Token(EQUAL,    "=",     null, line=1)
      1            вҶ’ Token(NUMBER,   "1",     1.0,  line=1)
       ;           вҶ’ Token(SEMICOLON,";",     null, line=1)
{                  вҶ’ Token(LEFT_BRACE,"{",   null, line=2)
  v a r            вҶ’ Token(VAR,      "var",   null, line=3)
    a              вҶ’ Token(IDENTIFIER,"a",    null, line=3)
      =            вҶ’ Token(EQUAL,    "=",     null, line=3)
        a          вҶ’ Token(IDENTIFIER,"a",    null, line=3)  вҶҗ жіЁж„ҸпјҡиҝҷжҳҜиҜ»еҸ–еҸҳйҮҸa
          +        вҶ’ Token(PLUS,     "+",     null, line=3)
            2      вҶ’ Token(NUMBER,   "2",     2.0,  line=3)
             ;     вҶ’ Token(SEMICOLON,";",     null, line=3)
  p r i n t        вҶ’ Token(PRINT,   "print", null, line=4)
        a          вҶ’ Token(IDENTIFIER,"a",    null, line=4)
         ;         вҶ’ Token(SEMICOLON,";",     null, line=4)
}                  вҶ’ Token(RIGHT_BRACE,"}",  null, line=5)

```

**Token еҲ—иЎЁе…ұ 17 дёӘпјҡ**
```
[VAR] [IDENTIFIER:a] [EQUAL] [NUMBER:1.0] [SEMICOLON]
[LEFT_BRACE]
[VAR] [IDENTIFIER:a] [EQUAL] [IDENTIFIER:a] [PLUS] [NUMBER:2.0] [SEMICOLON]
[PRINT] [IDENTIFIER:a] [SEMICOLON]
[RIGHT_BRACE]
[EOF]
```

## з¬¬дәҢйҳ¶ж®өпјҡParser иҜӯжі•и§Јжһҗ

**иҫ“е…Ҙпјҡ** Token еҲ—иЎЁ
**иҫ“еҮәпјҡ** ASTпјҲжҠҪиұЎиҜӯжі•ж ‘пјү

### и°ғз”Ёж Ҳеұ•ејҖиҝҮзЁӢ

```
parse()
в”ңв”Җв”Җ declaration()          вҶҗ ж¶Ҳиҙ№ VAR
в”Ӯ   в””в”Җв”Җ varDeclaration()
в”Ӯ       в”ңв”Җв”Җ consume(IDENTIFIER) вҶ’ "a"
в”Ӯ       в”ңв”Җв”Җ match(EQUAL) вҶ’ true
в”Ӯ       в””в”Җв”Җ expression()
в”Ӯ           в””в”Җв”Җ assignment()
в”Ӯ               в””в”Җв”Җ equality() вҶ’ comparison() вҶ’ term() вҶ’ factor() вҶ’ unary() вҶ’ primary()
в”Ӯ                   в””в”Җв”Җ match(NUMBER) вҶ’ Expr.Literal(1.0)
в”Ӯ       в””в”Җв”Җ consume(SEMICOLON)
в”Ӯ       в””в”Җв”Җ иҝ”еӣһ Stmt.Var(name=Token("a"), initializer=Expr.Literal(1.0))
в”Ӯ
в””в”Җв”Җ declaration()          вҶҗ дёҚжҳҜVARпјҢиҝӣе…Ҙ statement()
    в””в”Җв”Җ statement()
        в””в”Җв”Җ match(LEFT_BRACE) вҶ’ true вҶ’ block()
            в”ңв”Җв”Җ declaration()      вҶҗ ж¶Ҳиҙ№ VAR
            в”Ӯ   в””в”Җв”Җ varDeclaration()
            в”Ӯ       в”ңв”Җв”Җ consume(IDENTIFIER) вҶ’ "a"
            в”Ӯ       в”ңв”Җв”Җ match(EQUAL) вҶ’ true
            в”Ӯ       в””в”Җв”Җ expression()
            в”Ӯ           в””в”Җв”Җ assignment() вҶ’ equality() вҶ’ ... вҶ’ term()
            в”Ӯ               в”ңв”Җв”Җ factor() вҶ’ unary() вҶ’ primary()
            в”Ӯ               в”Ӯ   в””в”Җв”Җ match(IDENTIFIER) вҶ’ Expr.Variable(Token("a"))
            в”Ӯ               в”ңв”Җв”Җ match(PLUS) вҶ’ true
            в”Ӯ               в””в”Җв”Җ factor() вҶ’ unary() вҶ’ primary()
            в”Ӯ                   в””в”Җв”Җ match(NUMBER) вҶ’ Expr.Literal(2.0)
            в”Ӯ               в””в”Җв”Җ иҝ”еӣһ Expr.Binary(Expr.Variable("a"), +, Expr.Literal(2.0))
            в”Ӯ       в””в”Җв”Җ consume(SEMICOLON)
            в”Ӯ       в””в”Җв”Җ иҝ”еӣһ Stmt.Var(name=Token("a"), initializer=Expr.Binary(...))
            в”Ӯ
            в”ңв”Җв”Җ declaration()      вҶҗ дёҚжҳҜVARпјҢиҝӣе…Ҙ statement()
            в”Ӯ   в””в”Җв”Җ statement()
            в”Ӯ       в””в”Җв”Җ match(PRINT) вҶ’ true вҶ’ printStatement()
            в”Ӯ           в”ңв”Җв”Җ expression() вҶ’ ... вҶ’ primary()
            в”Ӯ           в”Ӯ   в””в”Җв”Җ match(IDENTIFIER) вҶ’ Expr.Variable(Token("a"))
            в”Ӯ           в””в”Җв”Җ consume(SEMICOLON)
            в”Ӯ           в””в”Җв”Җ иҝ”еӣһ Stmt.Print(Expr.Variable(Token("a")))
            в”Ӯ
            в”ңв”Җв”Җ check(RIGHT_BRACE) вҶ’ trueпјҢйҖҖеҮәеҫӘзҺҜ
            в””в”Җв”Җ consume(RIGHT_BRACE)
            в””в”Җв”Җ иҝ”еӣһ List[Stmt.Var(...), Stmt.Print(...)]
        в””в”Җв”Җ иҝ”еӣһ Stmt.Block(statements=[Stmt.Var(...), Stmt.Print(...)])
```

### Step 3пјҡжү§иЎҢеқ—еҶ… `Stmt.Var("a", Expr.Binary(Variable("a"), +, Literal(2.0)))`

**е…ій”®жӯҘйӘӨпјҡе…ҲеҜ№еҲқе§ӢеҢ–ејҸжұӮеҖјпјҢеҶҚе®ҡд№үеҸҳйҮҸ**

```
visitVarStmt(stmt)
    в”ңв”Җв”Җ stmt.initializer != null вҶ’ true
    в””в”Җв”Җ evaluate(Expr.Binary(...))
            в”Ӯ
            в–ј
        visitBinaryExpr
            в”ңв”Җв”Җ left = evaluate(Expr.Variable("a"))
            в”Ӯ           в”Ӯ
            в”Ӯ           в–ј
            в”Ӯ       visitVariableExpr
            в”Ӯ           в””в”Җв”Җ environment.get(Token("a"))
            в”Ӯ                   в”ңв”Җв”Җ local.values.containsKey("a") вҶ’ false вқҢ
            в”Ӯ                   в”ңв”Җв”Җ enclosing != null вҶ’ true
            в”Ӯ                   в””в”Җв”Җ enclosing.get(Token("a"))
            в”Ӯ                           в”ңв”Җв”Җ global.values.containsKey("a") вҶ’ true вң…
            в”Ӯ                           в””в”Җв”Җ иҝ”еӣһ 1.0
            в”Ӯ           в””в”Җв”Җ left = 1.0
            в”Ӯ
            в”ңв”Җв”Җ right = evaluate(Expr.Literal(2.0))
            в”Ӯ           в””в”Җв”Җ visitLiteralExpr вҶ’ иҝ”еӣһ 2.0
            в”Ӯ
            в”ңв”Җв”Җ operator: PLUS
            в””в”Җв”Җ 1.0 + 2.0 = 3.0  вҶ’  иҝ”еӣһ 3.0

    в””в”Җв”Җ environment.define("a", 3.0)   вҶҗ еңЁгҖҗеұҖйғЁзҺҜеўғгҖ‘дёӯе®ҡд№ү
            в””в”Җв”Җ local.values.put("a", 3.0)
```

> вҡ пёҸ **ж ёеҝғиҰҒзӮ№**пјҡеҲқе§ӢеҢ–ејҸ `a + 2` жұӮеҖјж—¶пјҢеұҖйғЁзҺҜеўғдёӯиҝҳжІЎжңү `a`пјҢ
> жүҖд»ҘжІҝ enclosing й“ҫжүҫеҲ°е…ЁеұҖзҺҜеўғзҡ„ `a = 1.0`пјҢи®Ўз®—еҫ— `3.0`гҖӮ
> д№ӢеҗҺжүҚеңЁеұҖйғЁзҺҜеўғдёӯе®ҡд№үж–°зҡ„ `a = 3.0`пјҢе®һзҺ°дәҶ**еҸҳйҮҸйҒ®и”ҪпјҲshadowingпјү**гҖӮ

**зҺҜеўғеҸҳеҢ–пјҡ**
```

в”Ӯ  еұҖйғЁзҺҜеўғ (local)            в”Ӯ  вҶҗ this.environment
в”Ӯ  enclosing: в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ

                                    в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
```

## е…ій”®зҹҘиҜҶзӮ№жҖ»з»“

### 1. еҸҳйҮҸйҒ®и”ҪпјҲShadowingпјү
еҶ…еұӮдҪңз”Ёеҹҹзҡ„ `var a` еҲӣе»әдәҶдёҖдёӘе…Ёж–°зҡ„еҸҳйҮҸпјҢиҰҶзӣ–дәҶеӨ–еұӮзҡ„ `a`пјҢдҪҶеӨ–еұӮ `a` дҫқз„¶еӯҳеңЁгҖӮ

```
е…ЁеұҖ:  a = 1.0   вҶҗ еқ—жү§иЎҢе®ҢеҗҺд»Қдёә 1.0
еқ—еҶ…:  a = 3.0   вҶҗ жҳҜеҸҰдёҖдёӘзӢ¬з«Ӣзҡ„еҸҳйҮҸпјҢеқ—з»“жқҹеҗҺж¶ҲдәЎ
```

### 2. еҲқе§ӢеҢ–ејҸжұӮеҖјж—¶жңә
`var a = a + 2` дёӯпјҢеҸідҫ§зҡ„ `a` еңЁ**еҪ“еүҚеұҖйғЁеҸҳйҮҸ `a` иў«е®ҡд№үд№ӢеүҚ**е°ұиў«жұӮеҖјпјҢеӣ жӯӨиҜ»еҸ–зҡ„жҳҜеӨ–еұӮдҪңз”Ёеҹҹзҡ„ `a = 1.0`пјҢиҖҢйқһиҮӘиә«гҖӮ

```
жү§иЎҢйЎәеәҸпјҡ
  1. е…ҲжұӮеҖј a + 2  вҶ’  жӯӨж—¶еұҖйғЁ a иҝҳдёҚеӯҳеңЁ  вҶ’  иҜ»еҸ–е…ЁеұҖ a = 1.0  вҶ’  з»“жһң 3.0
  2. еҶҚ define("a", 3.0)  вҶ’  еұҖйғЁ a жүҚиў«еҲӣе»ә
```

### 3. зҺҜеўғй“ҫжҹҘжүҫи§„еҲҷ
`Environment.get()` зҡ„жҹҘжүҫи·Ҝеҫ„пјҡ
```
еҪ“еүҚзҺҜеўғ вҶ’ enclosing(дёҠеұӮ) вҶ’ enclosing.enclosing(жӣҙдёҠеұӮ) вҶ’ ... вҶ’ global вҶ’ жүҫдёҚеҲ°еҲҷ RuntimeError
```

### 4. executeBlock зҡ„зҺҜеўғеҲҮжҚў
```java
void executeBlock(List<Stmt> statements, Environment environment) {
    Environment previous = this.environment;   // дҝқеӯҳж—§зҺҜеўғ
    try {
        this.environment = environment;        // еҲҮжҚўеҲ°ж–°зҺҜеўғ
        for (Stmt statement : statements) {
            execute(statement);
        }
    } finally {
        this.environment = previous;           // ж— и®әжҳҜеҗҰејӮеёёпјҢеҝ…йЎ»жҒўеӨҚ
    }
}
```
`finally` дҝқиҜҒеҚідҪҝеқ—еҶ…еҸ‘з”ҹиҝҗиЎҢж—¶й”ҷиҜҜпјҢзҺҜеўғд№ҹиғҪжӯЈзЎ®жҒўеӨҚгҖӮ
